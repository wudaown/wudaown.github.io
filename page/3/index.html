上次的爬虫写完之后，需要一个yunfile网盘的下载器.. 于是开始写..
过程不表.. 代码也暂时不贴.. 因为还没有完善..

可是 刚高兴没多久拿到windows安装依赖包就出现问题了…

马上就出现问题了好嘛！！

总结一下就是lxml包装不上.. 需要visual c++ 10.0 的编译器.. 然后这个版本没有64为的编译器.. 然后虽然有解决方案 不过超级麻烦啊…….

最后还是找到了事先编译好的lxml的whl包..

http://www.lfd.uci.edu/~gohlke/pythonlibs

其中yy是包名字xx是对应的python版本..

这样就顺利安装了..

PS:Mac下我都是用bpython的.. windows下的bpython有点问题 于是ipython也超好用的~~

作为一个救护人员 Medic

发表于 2014-12-05   |   分类于 部队   |  

作为一个Medic实习的几天

在部队被分配成了Medic.. 嘛.. 既然能学习新的知识那我就欣然接受了..
训练当然还是各种辛苦.. 毕竟是部队而不是学院什么的..

嘛.. 训练11月底基本结束了.. 于是部队把我们这群人丢到了救护车和医院去实习几天咯 ..
感触良多啊.. 接触到了很多的病患.. 每个人都有着各种不同的病症.. 也有车祸.. 跳楼等事件发生..

救护车的几天

基于职业素质这一点我就没有拍任何照片了.. 跟着救护车到处跑的几天.. 让我认识到了.. Medic所学的知识在和平时期对于多数的患者其实是无能为力的.. 我们更多的是给与病人和家属一种精神上的支持.. 病人、患者、伤者在疼痛的时候他们会显得非常的不知所措… 而我们就是用最快的速度赶到现场给予支援..

医院的几天

跟完了救护车下一个地方自然是医院了.. 在那里看到了各种手术.. 第一次接触的我来说.. 其实是非常的血腥和不适的..
为了能为患者从肺中插入一根管子以便呼吸，医生要切开他的胸口。真实的场景和我想象中的实在是差了太多了。

帮助医生吊盐水，为医生做简单的手术准备，为病人打疫苗，替医生观察病危的伤者的生命体征等等。
在医院也学了很多，一些简单的、基础的诊断知识。

更认识到了医生和护士是一个非常辛苦的行业 （最起码从我观察的几天来看 新加坡的确实是… -_-##

写在最后

跟着救护车的第一天我在生病，当然是感冒发烧等。
去的早上又下大雨，而且我偷懒没有带雨伞于是又是在雨中逗留了很久，结果报道的时候已经非常严重了。
可是实在 不想错过这个难得的机会，于是在救护车上自己给自己吊了两瓶盐水，有给自己配了很多的药。

结果完全没有效果啊！！！！！

AMD HD7950 刷 efi

发表于 2014-11-15   |   分类于 Mac   |  

#AMD HD7950 刷 efi

原来Mac Pro的显卡是GT120.. 多古老就不用我多说了吧..
自从换了27寸的Apple LED Display之后就感觉各种卡..

去SIM LIM问了一下熟悉的老黑Mac版本的7950开价$565新币… 太坑了.. 翻阅各种资料之后决定自己买一张公版的7950自己刷efi来实现开机的boot screen..
现在各种N卡和A卡都已经原生支持了.. 唯一缺的就是开机的boot screen而已.. 作为完美主义者 （+_+#) .. 当然是要实现的啦..

等了大概10天左右显卡终于从国内飘到新加坡了… 对了对了.. 一起来的还有两个定制的线..
Mac Pro的显卡电源线是小6pin转接6pin或者8pin的.. 新加坡买不到小6pin的头不然自己做了.. 6pin的头倒是还剩下很多..

插上显卡… 开机点亮了.. 如预料的没有 boot screen …

嘛.. 先不管了.. 重启进入bootcamp的WIN7备份一下显卡的bios ..
结果各种黑屏.. 好吧想到了WIN7的驱动是GT120的.. 于是换上GT120进入windows卸载nvidia显卡驱动..安装AMD催化剂咯…

再插上HD7950进入系统鸟.. 很开心啦 .. @_@+ ..

然后备份显卡的bios.. 这里说一下因为显卡支持UEFI的bios所有有一个1档和2档的开关于是就有了两个bios.. 建议都备份一下


这里就是开关了..
准备工作都做完了..

进入Mac系统.. 运行EFI ROM的制作工具..

mac_efi_create.zip

然后大家看README吧.. 里面写的很清楚鸟..
制作完ROM之后就是最后的步骤啦.. 刷！！！

进入windows打开ATI WINFLASH ..
载入ROM.. 刷 .. 保存.. 重启.. 祈祷！！

最后就变成这样啦..

写在最后.. 刷bios有风险.. 很有可能把显卡刷成砖.. 我的建议是各位买公版的7950原因是蓝宝石Mac版的7950的设计和公版的非常非常接近.. PCB等等..

PS:公版的7950加上国际运费加上两根定制线的一共是$190新币左右.. 如果买蓝宝石Mac版本的7950花费大概在$560++ ..
